使用 vue-cli 构建项目 eslint 报 Expected linebreaks to be 'LF' but found 'CRLF'错误.
解决方法：
进入`.eslintrc.js`文件，`rules`里添加`"linebreak-style": [0, "error", "windows"]`

```json
rules: {
    "linebreak-style": [0, "error", "windows"]
  }
```
